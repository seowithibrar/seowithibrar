# SEO Ibrar - Professional SEO Agency Website

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Deployed on Vercel (Static Site)

## Local Development
Open index.html in browser or use Live Server extension

## Deployment
Push to GitHub → Connect to Vercel → Auto-deploys on push

## URL Structure
- / → Home
- /about → About
- /services → All Services
- /services/seo → SEO Services
- /services/seo-lahore → SEO Services Lahore
- /services/seo-pakistan → SEO Services Pakistan
- /services/seo-education → Education SEO
- /services/seo-visa → Visa SEO
- /services/seo-wordpress → WordPress Solutions
- /services/seo-wordpress-education → WordPress Education
- /services/seo-wordpress-visa → WordPress Visa
- /portfolio → Portfolio
- /contact → Contact
